## PART_01. 기초 문법

### JavaScript란?
| 역할 | 담당 |
|---|---|
| HTML | 뼈대 ("페이지에 버튼이 있다") |
| CSS | 스타일 ("버튼은 파란색 둥근 모양이다") |
| JavaScript | 동작 ("버튼을 클릭하면 ○○이 일어난다") |

- **콘솔**: F12 → Console 탭. JS 즉시 실행/디버깅 공간.

### 변수
- **선언**: 공간을 만드는 것
- **할당**: 값을 넣는 것
- **선언+할당**: 동시에 하는 것

```js
let name = "홍길동";
let age = 20;
```

| 키워드 | 특징 |
|---|---|
| `let` | 변경 가능한 변수 |
| `const` | 변경 불가능한 상수 (기본값, 권장) |

> `var`는 옛 방식(블록 스코프 없음, 재선언 가능) — 지금은 쓰지 않는다.

### 자료형
| 자료형 | 예시 | 설명 |
|---|---|---|
| string | `"안녕"`, `` `템플릿` `` | 텍스트 |
| number | `42`, `3.14`, `-10` | 정수/소수 |
| boolean | `true`, `false` | 참/거짓 |
| undefined | `undefined` | 값이 아직 없음 |
| null | `null` | 값이 없음(명시적) |
| array | `[1,2,3]` | 순서 있는 목록 |
| object | `{ name: "홍길동" }` | 키-값 쌍 데이터 |

- 템플릿 리터럴: 백틱(`` ` ``) + `${변수}` → `` `이름: ${name}살` ``
- 문자열 메서드: `.length`, `.toUpperCase()`, `.trim()`, `.includes("x")`

### 연산자
- 산술: `+ - * / % **`
- ⚠️ `"5" + 3 = "53"` (문자열+숫자=문자열 연결) → 계산 전 `Number()` 변환 필수
- 비교: `==`(자료형 무시) vs `===`(값+자료형 모두 같음) → **항상 `===` 사용**
- 논리: `&&`(AND), `||`(OR), `!`(NOT)

### 조건문
```js
if (score >= 90) { ... }
else if (score >= 80) { ... }
else { ... }
```
- 삼항 연산자: `조건 ? 참일때값 : 거짓일때값` — 단순 분기에 적합

### 반복문
- **for**: `for (초기값; 조건; 증감) { ... }`
- **while**: 조건만으로 도는 반복문
- **do-while**: 본문을 먼저 한 번 실행 후 조건 확인
- 누적 계산: `sum += i` (= `sum = sum + i`)

## PART_02. 배열·객체·함수

### 배열
```js
const fruits = ["사과", "바나나", "딸기"];
fruits[0]        // "사과" (인덱스 0부터)
fruits.length     // 3
fruits.push("포도")  // 끝에 추가
fruits.pop()      // 마지막 요소 제거
```
- for문 순회: `for (let i=0; i<fruits.length; i++) { fruits[i] }`

### 객체
```js
const user = { name: "홍길동", age: 20, email: "..." };
user.name        // 점 표기법
user["age"]      // 대괄호 표기법
user.city = "서울"; // 새 키 추가
```

### 함수
```js
function greet(name) {
  console.log(`안녕하세요, ${name}님!`);
}
```
- **매개변수**: 입력값 / **반환값(return)**: 결과값
- `return` 만나면 함수 즉시 종료, 없으면 `undefined` 반환

**함수 표현식 vs 화살표 함수**
| 구분 | this |
|---|---|
| 일반 함수 | 호출 주체에 따라 동적 |
| 화살표 함수 | 선언된 위치(상위 스코프) 고정 |

- 화살표 함수는 `this`가 고정되므로 콜백(배열 메서드, 이벤트타이머)에 적합
- 함수 표현식은 객체 메서드, `new` 인스턴스 생성에 적합
- 함수 선언식은 호이스팅 O, 화살표/표현식은 호이스팅 X (선언 전 호출 불가)

### 스코프
- **블록 스코프**: `{}` 안에서만 유효 (let/const)
- **함수 스코프**: 함수 안에서만 유효
- **전역 변수**: 함수 밖 선언 → 어디서든 접근
- **지역 변수**: 함수 안 선언 → 그 안에서만 접근
- 원칙: 필요한 범위 안에서만 선언하자 (전역 남발 금지)

## PART_03. HTML + JS 연결

- `<script>` 태그: `</body>` 바로 앞 배치 (또는 `<head>` + `defer` 속성)
- `defer`: HTML 파싱과 함께 JS 다운로드, HTML 다 읽은 후 실행 → head에 깔끔히 둘 수 있음
- 경로: `../` = 상위 폴더로 이동

### querySelector
```js
document.querySelector("#id")     // #id
document.querySelector(".class")  // .class
document.querySelector("tag")     // 태그명
```
- 요소 내용 변경: `result.innerHTML = "결과: 42"`

### 이벤트
```js
const input = document.querySelector("#num1");
input.value  // 입력값 (항상 문자열로 반환)

btn.addEventListener("click", function() {
  // 클릭 시 실행할 코드
});
```

## PART_04. 실습

### 실습① 메뉴판 만들기
- `menu` 배열(객체 여러 개) → `menuList` `<ul>`에 반복문으로 채워 넣기
```js
const menu = [
  { name: "아메리카노", price: 4000 },
  { name: "카페라떼", price: 4500 },
  { name: "바닐라라떼", price: 5000 },
];
let html = "";
for (let i = 0; i < menu.length; i++) {
  html += `<li><span>${menu[i].name}</span><span>${menu[i].price}원</span></li>`;
}
menuList.innerHTML = html;
```
- 도전과제: 메뉴 2개 추가 / `category` 속성 추가해 "이름-가격(분류)" 표시 / 전체 합계 계산해 "총 합계: ○○원" 표시
