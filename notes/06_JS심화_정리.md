## PART_01 · DOM 선택과 조작

> 브라우저가 HTML을 읽어 만든 트리가 **DOM**. JS는 이 트리를 읽고 고친다.

### DOM 트리
```text
document
└ body
   ├ h1   "제목"
   ├ p    "내용"
   └ div
      └ span "텍스트"
```

### 요소 선택 — select.js
```js
document.querySelector("선택자")     // 첫 1개
document.querySelectorAll("선택자")  // 모두

const title = document.querySelector("h1");
const btn   = document.querySelector("#submitBtn");
const cards = document.querySelectorAll(".card");
```
- 선택자는 CSS 문법 그대로 (`태그` · `#id` · `.class`)

### textContent vs innerHTML — content.js
```js
const el = document.querySelector("#r");

el.textContent = "<b>굵게</b>";  // <b>굵게</b> 글자 그대로 표시
el.innerHTML   = "<b>굵게</b>";  // 굵게 — 실제로 굵게 렌더링
```
| 속성 | 태그 해석 | 용도 |
|------|-----------|------|
| `textContent` | ❌ 글자 그대로 | 순수 텍스트 넣을 때 (안전) |
| `innerHTML` | ⭕ HTML로 렌더링 | 태그째로 넣을 때 |

### 요소 생성 — create.js
```js
const p = document.createElement("p");
p.textContent = "새로 만든 단락";

const btn = document.createElement("button");
btn.textContent = "클릭";
btn.classList.add("btn-primary");

// DOM에 추가해야 화면에 나타난다
document.querySelector("#box").appendChild(p);
```
> `createElement`만 하면 메모리에만 존재 — **`appendChild`로 붙여야 화면에 보인다** ⭐

### 추가와 제거 — list.js
```js
const list = document.querySelector("#list");
const item = document.createElement("li");
item.textContent = "새 항목";
list.appendChild(item);   // 끝에 추가

const first = document.querySelector("li");
first.remove();           // 첫 번째 li 제거
```

## PART_02 · 요소 삽입과 복제

### insertAdjacentHTML / insertAdjacentElement — insert.js
```js
const box = document.querySelector("#box");

box.insertAdjacentHTML("beforeend",
  "<p>안쪽 맨 뒤에 추가</p>");

const tag = document.createElement("span");
box.insertAdjacentElement("afterbegin", tag);
```
| 위치 값 | 삽입 지점 |
|---------|-----------|
| `beforebegin` | 요소 바로 앞 (바깥) |
| `afterbegin` | 요소 안쪽 맨 앞 |
| `beforeend` | 요소 안쪽 맨 뒤 |
| `afterend` | 요소 바로 뒤 (바깥) |

- `insertAdjacentHTML` — HTML **문자열**을 원하는 위치에
- `insertAdjacentElement` — 만들어 둔 **요소**를 원하는 위치에

### 복제 — clone.js
```js
const card = document.querySelector(".card");
const copy = card.cloneNode(true);  // 내부 내용까지 통째로 복사

document.querySelector("#list")
  .appendChild(copy);
```
> `cloneNode(true)` = 자식·텍스트까지 깊은 복사. 복제한 것도 **appendChild 해야 화면에 등장**

## PART_03 · 이벤트 심화

### 기본형 — event.js
```js
const btn = document.querySelector("#btn");

btn.addEventListener("click", function() {
  console.log("클릭됨");
});
```

### 이벤트 객체 — target.js
```js
btn.addEventListener("click",
  function(event) {
    console.log(event.target);  // 실제로 클릭된 요소
    console.log(event.type);    // "click"
  });
```
| 속성/메서드 | 의미 |
|-------------|------|
| `event.target` | **실제로 이벤트가 일어난** 요소 |
| `event.currentTarget` | **리스너가 붙어 있는** 요소 |
| `event.type` | 이벤트 이름 (`"click"` 등) |
| `event.preventDefault()` | 기본 동작 취소 |

### 폼 제출 — 새로고침 방지
```js
form.addEventListener("submit",
  function(event) {
    event.preventDefault();
    // 새로고침 방지
    // 여기서 직접 처리
  });
```

### input — 실시간 감지
```js
input.addEventListener("input",
  function(event) {
    console.log(event.target.value);
    // 입력 중인 값
  });
```

### 버블링 — bubbling.js
```js
const ids = ["div1","div2","div3","div4"];
ids.forEach(function(id) {
  document.querySelector("#" + id)
    .addEventListener("click", function(e) {
      console.log("실행:", e.currentTarget.id);
    });
});
```
- 안쪽 요소를 클릭하면 이벤트가 **안 → 밖으로 전파**되며 조상들의 리스너도 차례로 실행
- 가장 안쪽 `div4` 클릭 → `div4 → div3 → div2 → div1` 순서로 로그

### 이벤트 위임 — delegation.js ⭐
```js
const list = document.querySelector("#list");

// 부모 하나에만 리스너
list.addEventListener("click", function(e) {
  if (e.target.classList.contains("del")) {
    e.target.closest("li").remove();
  }
});
```
- 자식마다 리스너를 붙이지 않고 **부모 하나**로 처리 → 나중에 추가된 자식도 자동으로 동작
- `closest("선택자")` — 자기 자신부터 조상 방향으로 가장 가까운 일치 요소 탐색

### 바깥 클릭 감지 — outside.js
```js
document.addEventListener("click",
  function(e) {
    if (!e.target.closest(".dropdown")) {
      menu.classList.remove("open");
      // 바깥 클릭 → 닫기
    }
  });
```
> `document`에 리스너 + `closest`로 "내부인지" 판별 — 드롭다운·모달 닫기의 정석 패턴

## PART_04 · matchMedia — JS로 반응형 감지

> CSS 미디어 쿼리를 **JS에서** 사용. 화면 크기에 따라 로직을 분기할 수 있다.

### 현재 상태 확인 — match.js
```js
const mq = window.matchMedia(
  "(max-width: 768px)");

if (mq.matches) {
  console.log("모바일 화면");
} else {
  console.log("데스크톱 화면");
}
```

### 변경 감지 — change.js
```js
const mq = window.matchMedia("(max-width: 768px)");

mq.addEventListener("change", function(e) {
  if (e.matches) menu.classList.add("mobile");
  else menu.classList.remove("mobile");
});
```
- `mq.matches` — 지금 조건에 맞는지 (boolean)
- `"change"` 이벤트 — 경계(768px)를 **넘나드는 순간**마다 실행

## PART_05 · 웹 컴포넌트

### 왜 필요한가
| 복사해서 붙이면 | 컴포넌트로 묶으면 |
|-----------------|-------------------|
| `a.html` · `b.html` · `c.html` 각각 `<header>…</header>` 복사 | 세 파일 모두 `<my-header></my-header>` 한 줄 |
| 하나 고치면 **세 곳 모두** 손봐야 한다 | 정의 **한 곳만** 고치면 전부 반영 |

### 기본형 — my-header.js
```js
class MyHeader extends HTMLElement {
  connectedCallback() {  // 화면에 붙는 순간
    this.innerHTML = `
      <header>
        <h1>내 사이트</h1>
      </header>`;
  }
}
// 태그 이름과 class 연결
customElements.define("my-header", MyHeader);
```
- 커스텀 태그 이름에는 **하이픈(-)이 필수** (`my-header`, `my-card`)

### 사용 — index.html
```html
<body>
  <my-header></my-header>
  <main>…본문…</main>
  <my-footer></my-footer>
  <script src="my-header.js"></script>
</body>
```

### 속성 받기 — my-card.js
```js
connectedCallback() {
  const t = this.getAttribute("title");
  const d = this.getAttribute("desc");
  this.innerHTML = `
    <div class="card">
      <h3>${t}</h3><p>${d}</p>
    </div>`;
}
```
`<my-card title="..." desc="...">` 처럼 HTML 속성으로 데이터를 넘긴다

### 생명주기 콜백
| 콜백 | 호출 시점 | 용도 |
|------|-----------|------|
| `connectedCallback()` | 요소가 화면에 **추가될 때** | 보통 여기서 내용을 그린다 (주로 사용) ⭐ |
| `disconnectedCallback()` | 요소가 화면에서 **제거될 때** | 정리 작업에 쓴다 (선택) |

## 🛠 실습

### 실습① — 카드 보드 (insert)
입력값으로 카드를 만들어 **맨 앞/맨 뒤 추가 · 마지막 카드 복제**

```html
<!-- src/html/insert.html -->
<div class="board">
  <input id="text" placeholder="카드 내용">
  <button id="addTop">맨 앞에 추가</button>
  <button id="addBottom">맨 뒤에 추가</button>
  <button id="dup">마지막 카드 복제</button>
  <div id="list">
    <div class="card">샘플 카드</div>
  </div>
</div>
<script src="../js/insert.js"></script>
```
```js
// src/js/insert.js
const text = document.querySelector("#text");
const list = document.querySelector("#list");

function makeCard() {
  return `<div class="card">${text.value}</div>`;
}

document.querySelector("#addTop")
  .addEventListener("click", function() {
    list.insertAdjacentHTML("afterbegin", makeCard());  // 맨 앞
  });

document.querySelector("#addBottom")
  .addEventListener("click", function() {
    list.insertAdjacentHTML("beforeend", makeCard());   // 맨 뒤
  });

document.querySelector("#dup")
  .addEventListener("click", function() {
    const cards = document.querySelectorAll(".card");
    const last = cards[cards.length - 1];  // 마지막
    const copy = last.cloneNode(true);     // 복제
    list.appendChild(copy);
  });
```
- CSS: `.board { max-width: 480px; margin: 0 auto; }` · `.card { padding · border · border-radius: 8px · background: #f8f9fa }`

### 실습③ — 투두 리스트
```html
<!-- src/html/todo.html -->
<div class="todo">
  <div class="form">
    <input id="text" placeholder="할 일">
    <button id="add">추가</button>
  </div>
  <ul id="list"></ul>  <!-- 비워둔다 -->
</div>
```
```js
// src/js/todo.js — 추가
const text = document.querySelector("#text");
const list = document.querySelector("#list");

function createLi(value) {
  const li = document.createElement("li");
  const label = document.createElement("span");
  label.className = "label"; label.textContent = value;
  const del = document.createElement("button");
  del.className = "del"; del.textContent = "×";
  li.append(label, del);  // 한 번에 추가
  return li;
}

document.querySelector("#add").addEventListener("click", () => {
  if (text.value.trim() === "") return;  // 빈값 무시
  list.appendChild(createLi(text.value));
  text.value = "";  // 입력창 비우기
});
```
```js
// todo.js — 위임 처리
list.addEventListener("click", function(e) {
  const li = e.target.closest("li");
  if (!li) return;
  if (e.target.classList.contains("del")) {
    li.remove();                      // 삭제 버튼 → 항목 제거
  } else {
    li.classList.toggle("done");      // 그 외 → 완료 토글
  }
});
```
- CSS: `#list li { display: flex; justify-content: space-between; align-items: center; }` · `.done .label { text-decoration: line-through; color: #bbb; }` · `.del { border: none; background: none; color: #e25555; cursor: pointer; }`

### 실습④ — 웹 컴포넌트 포트폴리오
`createElement`만으로 헤더 컴포넌트 조립 (innerHTML 없이):
```js
// src/js/MyHeader.js
class MyHeader extends HTMLElement {
  connectedCallback() {
    const header = document.createElement("header");
    const h1 = document.createElement("h1");
    h1.textContent = "내 포트폴리오";
    const nav = document.createElement("nav");
    const links = [
      { text: "소개",  href: "./portfolio.html" },
      { text: "할 일", href: "./todo.html" },
    ];
    links.forEach(function(link) {
      const a = document.createElement("a");
      a.textContent = link.text;
      a.href = link.href;
      nav.appendChild(a);
    });
    header.appendChild(h1);
    header.appendChild(nav);
    this.appendChild(header);  // 컴포넌트에 붙임
  }
}
customElements.define("my-header", MyHeader);
```
```js

class MyFooter extends HTMLElement {

    footer.appendChild(p);
    this.appendChild(footer);
  }
}
customElements.define("my-footer", MyFooter);
```
```js
// src/js/MyCard.js — 빈 칸을 채운다 (과제)
```

## ✅ 6강 정리
- DOM은 브라우저가 HTML로 만든 트리 — `querySelector`로 잡고 `createElement + appendChild`로 만들어 붙인다
- `insertAdjacentHTML`은 4가지 위치(`beforebegin · afterbegin · beforeend · afterend`)에 삽입, `cloneNode(true)`는 통째로 복제
- 이벤트는 버블링으로 위로 전파 → 부모 하나에 리스너를 두는 **이벤트 위임**(`closest` + `classList.contains`)이 실전 패턴
- `document` 리스너 + `!closest(".dropdown")`로 바깥 클릭 닫기, `matchMedia`로 JS에서 화면 크기 분기
- 반복되는 UI는 `class ~ extends HTMLElement` + `connectedCallback` + `customElements.define`으로 커스텀 태그 하나로 묶는다
